Grow a Garden API

API cung cấp dữ liệu cho trò chơi Roblox Grow a Garden

Phiên bản API: 3.5.9
Đặc tả: OAS 3.1
Giấy phép: MIT

Về API

API này cung cấp dữ liệu về Hạt giống, Trang bị, Trứng, Vật phẩm trang trí và Thời tiết từ trò chơi Grow a Garden Roblox game.

Tất cả các endpoint trả về dữ liệu dưới dạng JSON và tuân theo quy ước REST.
